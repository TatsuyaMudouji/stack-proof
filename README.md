# stack-proof

同一の業務ドメイン（estimate: 面積×単価）を複数言語・複数実行環境で実装し、
設計・入力検証・テスト・実行方法を揃えたサンプル集です。

## Modules
- Node.js / TypeScript: ./node-ts
- Go: ./go
- PHP: ./php
- Ruby: ./ruby
- Java: ./java
- Swift: ./swift
- C (or C++): ./c
- HTML/CSS: ./html-css

## Conventions
- 「入力検証（validation）」を必ず含める
- 各ディレクトリに実行方法（RUN）を記載する
- 可能な範囲でテストを用意する
